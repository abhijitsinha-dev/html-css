# Flipping Card Project

A 3D flipping credit card built with HTML and CSS featuring a smooth hover flip animation, glassmorphism styling, and realistic front/back card details.

## Features

- 3D flip animation on hover
- Glassmorphism card style (blurred, translucent background)
- Front side with logo, chip, number, name, and expiry
- Back side with magnetic strip, CVV, and info text
- Responsive layout with centered presentation

## Technologies Used

- HTML5
- CSS3 (Flexbox, 3D transforms, backdrop-filter)
- Normalize.css for cross-browser consistency

## How to Use

1. Open `index.html` in your browser.
2. Replace images in `images/` (e.g., `logo.png`, `chip.png`) if you want to customize branding.
3. Modify text content in `index.html` (card number, name, dates) to fit your needs.

## Customization

- Colors, backgrounds, and card appearance can be changed in `style.css`.
- Flip behavior and perspective can be tuned via `.container` and `section` styles.
