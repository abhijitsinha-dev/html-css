# Profile Card Project

A stylish and modern profile card built with HTML and CSS featuring social media links, user analytics, and attractive UI.

## Features

- User name & job/title display
- Circular profile picture section
- Social media links with icons (Font Awesome)
- Action buttons for "Subscribe" and "Message"
- Analytics section (Likes, Comments, Shares)
- Responsive and modern design

## Technologies Used

- HTML5
- CSS3 (including Flexbox)
- Font Awesome (for icons)

## How to Use

1. Clone or download this repository/project folder.
2. Open `index.html` in your browser.
3. Replace `profile-pic.jpg` in the folder with your desired profile picture for personalization.
4. Modify the HTML (name, job, social links, etc.) to fit your needs.

## Customization

- Colors and fonts can be changed via `style.css`.
- Social links and button actions can be changed in `index.html`.
